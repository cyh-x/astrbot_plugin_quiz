# Quiz 知识问答插件

一个基于 [FoxQuiz](https://foxquiz.com) 网站的知识问答插件，适用于 AstrBot 框架。  
支持多种题库类型（如 `naruto`, `marvel`, `bleach` 等），可下载题库、进行问答游戏，并记录用户的累计奖金和最高纪录。

## 功能特性

- **题库管理**  
  - 使用 `/quiz_load <题型>` 从 FoxQuiz 网站下载指定类型的题库（包括题目、选项、答案和图片）。  
  - 使用 `/quiz_unload <题型>` 删除已下载的题库。

- **知识问答游戏**  
  - 输入 `/quiz <题型>` 开始游戏，随机抽取 10 道题目。  
  - 每答对一题奖金递增（第1题 50 元，第2题 100 元，……，第10题 500 元）。  
  - 答错或超时则奖金清零，游戏结束。  
  - 可中途输入“退出”或“quit”带走已获得的奖金。  
  - 游戏结束后自动更新用户的累计总奖金和该题型最高纪录。

- **统计与排行榜**  
  - 使用 `/quiz_stats get` 查看自己的累计奖金和最高纪录。  
  - 使用 `/quiz_stats rank [题型]` 查看所有用户的总奖金排行榜（支持按全部类型或指定题型统计）。

- **跨平台支持**  
  基于 AstrBot 通用接口开发，支持所有 AstrBot 已适配的消息平台（包括 QQ、Telegram、微信、Discord 等）。

## 安装与配置

1. 将插件放置于 AstrBot 的 `data/plugins/` 目录下。
2. 安装依赖（若自动安装失败，请手动执行）：
   ```bash
   pip install aiohttp beautifulsoup4
   ```
3. 重启 AstrBot，插件将自动加载。

## 使用方法

### 下载题库

```bash
/quiz_load <题型>
```

示例：  
`/quiz_load naruto` — 下载火影忍者题库

**说明**  
- 题库文件将保存在 `AstrBot/data/plugin_data/astrbot_plugin_quiz/` 目录下。  
- 包含一个 JSON 文件（题目数据）和一个以 `_quiz_images` 结尾的图片文件夹。

### 卸载题库

```bash
/quiz_unload <题型>
```

删除指定题型的所有相关文件（JSON、HTML、图片文件夹）。

### 开始游戏

```bash
/quiz <题型>
```

游戏规则：  
- 共 10 题，每道题作答时间 15 秒。  
- 答对后奖金累加，并询问是否继续（输入除“退出/quit”外的任意内容继续）。  
- 答错或超时游戏结束，奖金清零（但累计总奖金不受影响）。  
- 全部答对可获得全额奖金（共计 2750 元）。

### 查看统计

#### 个人统计

```bash
/quiz_stats get
```

显示当前用户在所有题型下的累计总奖金和各题型的最高纪录。

#### 排行榜

```bash
/quiz_stats rank [题型]
```

- 不加参数时默认显示所有题型的总奖金排行榜。  
- 指定题型（如 `naruto`）则仅显示该题型的累计总奖金排行榜。

### 列出已下载的题库

```bash
/quiz_list
```

返回当前已下载且可用的题库名称列表。

## 数据存储

所有数据均存放在 `AstrBot/data/plugin_data/astrbot_plugin_quiz/` 目录下：

- `user_data.json` — 用户统计信息（累计奖金、最高纪录等）。  
- `{题型}_quiz.json` — 题目数据。  
- `{题型}_quiz_images/` — 对应题目的图片文件。

## 注意事项

### 关于 FoxQuiz 网站的许可

- FoxQuiz 官方声明：通过其服务生成的内容在 **知识共享零 (CC0) 许可** 下提供，通常可自由用于任何目的（包括商业用途），无需署名。  
- 但生成的输出中可能包含第三方权利（如图片、角色名称等），用户需自行确保使用不侵犯第三方知识产权。  
- 本插件仅作为工具提供题库下载和游戏功能，开发者不对用户使用这些内容的行为承担法律责任。

### 图片下载不稳定

- FoxQuiz 网站的部分图片可能无法正确加载（尤其是一些新题库或网络波动时）。  
- 若下载图片失败，答题时可能缺少对应图片（插件会跳过图片仅发送文本）。  
- **解决办法**：可以尝试过一段时间重新下载（先 `/quiz_unload` 再 `/quiz_load`），或选择图片需求不高的题库直接使用。

### 并发游戏

- 当前版本未对同一用户同时进行多个游戏进行隔离，请避免在多个会话中同时开始同一题库的游戏，以免状态混乱。  
- 建议每个用户只在一个会话中进行游戏。
